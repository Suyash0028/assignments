Question 1:

Which of the following is not a type of database model?

a) Hierarchical

b) Relational

c) Linear

d) NoSQL

Answer: c) Linear

Question 2:

What is the purpose of the SQL SELECT statement?

a) To insert data into a table

b) To update data in a table

c) To retrieve data from a table

d) To delete data from a table

Answer: c) To retrieve data from a table

Question 3:

Which of the following is a primary key in a relational database?

a) Duplicate values allowed

b) Unique identifier for a record

c) Stores only numeric values

d) Case-sensitive text

Answer: b) Unique identifier for a record

Question 4:

What does the acronym SQL stand for?

a) Structured Query Language

b) Simple Question Language

c) Sequential Query Language

d) Systematic Question Language

Answer: a) Structured Query Language

Question 5:

In a relational database, what is a foreign key?

a) A key that is shared among multiple tables

b) A key that uniquely identifies each record in a table

c) A key that is automatically generated by the database

d) A key that links to the primary key in another table

Answer: d) A key that links to the primary key in another table

Question 6:

What is normalization in the context of databases?

a) Adding redundancy to improve performance

b) Organizing data to minimize redundancy and dependency

c) Combining multiple tables into one for simplicity

d) Deleting unnecessary data

Answer: b) Organizing data to minimize redundancy and dependency

Question 7:

Which of the following is not a relational database management system (RDBMS)?

a) MySQL

b) Oracle

c) MongoDB

d) SQL Server

Answer: c) MongoDB

Question 8:

What is the purpose of the SQL WHERE clause?

a) To filter data based on specified conditions

b) To order the results of a query

c) To perform mathematical calculations

d) To join multiple tables

Answer: a) To filter data based on specified conditions

Question 9:

Which of the following statements is used to add new rows to a table in SQL?

a) SELECT

b) UPDATE

c) INSERT

d) DELETE

Answer: c) INSERT

Question 10:

Which of the following is not a valid data type in SQL?

a) VARCHAR

b) INTEGER

c) BOOLEAN

d) FLOATING-POINT

Answer: c) BOOLEAN
